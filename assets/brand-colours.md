# CxB Brand Colours

## Core palette

- **Navy (primary)**
  - Hex: `#0A2342`
  - Usage: headers, footer, primary text on dark, backgrounds for strong sections.

- **Teal (accent)**
  - Hex: `#00B7B2`
  - Usage: highlights, buttons, logo outline, radar chart stroke.

- **Amber (warning / developing)**
  - Hex: `#FFB74D`
  - Usage: warning / developing maturity accents (optional).

- **Red (low maturity / risk)**
  - Hex: `#D32F2F`
  - Usage: low maturity badges, risk flags.

- **Background grey**
  - Hex: `#F5F7FA`
  - Usage: page background, separating cards from background.

- **Muted text**
  - Hex: `#6B7280`
  - Usage: subtitles, helper text, explanatory copy.

## Typography

- **Primary stack:** system UI
  - `system-ui, -apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif`
- **Logo style:** rounded sans‑serif (Regular weight), Nunito‑like

## Components

- Cards: white (`#FFFFFF`) with soft shadow (`0 2px 8px rgba(0,0,0,0.08)`)
- Border radius: `8px`
- Buttons:
  - Primary: teal background, dark teal text
  - Secondary: light grey background, dark text

These tokens are implemented in `style.css` and can be reused across other CxB tools and pages.
